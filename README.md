# WAW (Windows and Windows on Windows)

## 🚀 Что это?
WAW — это подсистема для Windows 10 (и выше), которая позволяет запускать **16-битные и 8-битные программы** прямо из современной ОС.

## 🎯 Зачем?
Microsoft убила поддержку 16-битных приложений в 64-битных Windows. WAW возвращает её обратно!

## 🔧 Как работает
- Эмуляция процессора Intel 8086
- Поддержка .COM и .EXE файлов
- Интеграция с папкой C:\Windows\SysWOW16
- Полная совместимость с MS-DOS программами

## 📥 Установка
1. Скачайте последний релиз
2. Запустите install.bat от имени администратора
3. Копируйте старые программы в C:\Windows\SysWOW16
4. Запускайте из командной строки: `COMMAND`

## 🕊️ Поддержка 8-бит (в разработке)
Скоро добавим поддержку CP/M и старых 8-битных систем!
